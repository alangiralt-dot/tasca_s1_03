# S1.03 - PHP Arrays & Collections

This repository contains the solutions for the PHP Arrays exercise block (S1.03). The objective is to practice array manipulation, associative structures, and functional programming techniques like `array_map`, `array_filter`, and `array_reduce`.

## Summary of Exercises

### Level 1: Basics
- **Exercise 1:** Creating a simple integer array and iterating through it.
- **Exercise 2:** Measuring array size and deleting elements.
- **Exercise 3:** Logic to verify if a specific character exists across all strings in an array.
- **Exercise 4:** Creating and accessing associative arrays using personal data keys.

### Level 2: Advanced Manipulation
- **Exercise 1:** Performing set operations (intersections, merges, and differences) between two lists of strings.
- **Exercise 2:** Managing complex associative arrays (nested structures) to calculate individual and group averages.

### Level 3: Higher-Order Functions
- **Exercise 1:** Using `array_map()` to transform an array (calculating cubes).
- **Exercise 2:** Using `array_filter()` to extract strings based on length parity.
- **Exercise 3:** Using `array_reduce()` and custom logic to sum only prime numbers within an unordered collection.
